Fake News Detection System

Final Year Project — Python, Flask, Machine Learning

Project Overview

Machine learning and NLP system classifying news as real or fake. Built a Flask web server with REST endpoints and a simple HTML interface.

Key Features

  • Logistic Regression with TF-IDF & DistilBERT models
  • Flask API with REST endpoints
  • Hands-on NLP preprocessing, model training, evaluation

Tech Stack & Dataset

Python, Flask, scikit-learn, Hugging Face Transformers, PyTorch, Pandas, NLTK

Dataset: Kaggle Fake and Real News Dataset
