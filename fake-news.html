Fake News Detection System

Final Year Project — Python, Flask, Machine Learning

Project Overview

The project was to see whether Machine learning models or Deep learning models would be best to detect Fake news or real news.

In the end it was determined that Machine learning and NLP system classifying news as real or fake was best and would then be deployed by a Flask web server with REST endpoints and a simple HTML interface.

Key Features

  • Logistic Regression with TF-IDF & DistilBERT models
  • Flask API with REST endpoints
  • Hands-on NLP preprocessing, model training, evaluation

Tech Stack & Dataset

Python, scikit-learn, Hugging Face Transformers, PyTorch, Pandas, NLTK, Flask

Dataset: Kaggle Fake and Real News Dataset
