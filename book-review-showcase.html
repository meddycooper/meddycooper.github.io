Book Review Platform

Advanced Web Development Assignment — Symfony, JWT API, Google Books Integration

This project is a full-stack web application developed in Symfony that allows users to browse, review, and manage books. It includes features such as user authentication, profile management, image uploads, advanced search filters, and integration with the Google Books API. The admin dashboard provides role-based controls for managing users, reviews, and content.
