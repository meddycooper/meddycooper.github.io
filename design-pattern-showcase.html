JavaFX Design Pattern Educational Game

Design Patterns Assignment — JavaFX, Delegation, Strategy, Observer Patterns

This project was developed as part of the Design Patterns module and demonstrates the use of object-oriented design and multiple software design patterns in a JavaFX-based educational game. The game teaches children about wildlife habitats and the butterfly lifecycle through interactive canvas-based animations and a quiz. This project is more about the code than the game itself. I used images from the internet to create the game.

Built using Java and JavaFX in Eclipse with Maven, the game applies Delegation, Strategy, and Observer patterns to manage interactions, object behaviour, and user interface updates. The architecture allows for easy extension and addition of new animals, questions, or learning scenarios.

The screenshots below showcase key aspects of the project — from the interactive map and information messages to the butterfly lifecycle simulation and quiz feature.

Key Design Features

  • Delegation Pattern: Used in the Egg class to transition between growth stages.
  • Strategy Pattern: Enables interchangeable behaviour for different butterfly lifecycle stages.
  • Observer Pattern: Zoo class observes user interactions and updates knowledge counters.
  • Extensible Architecture: Easily add new animals, habitats, or lifecycle stages using interfaces and inheritance.
  • Educational Focus: Combines interactivity, visual learning, and reinforcement through quizzes.
